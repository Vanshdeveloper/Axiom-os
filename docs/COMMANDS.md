# 🤖 Axiom OS Telegram Commands

Axiom OS gives you absolute control over your productivity directly from Telegram. Below is the official registry of all supported commands.

## 📋 Task & Backlog Management
*Integrated with your Notion Backlog database.*

| Command | Action | Example |
| :--- | :--- | :--- |
| `/add [task]` | Quick-add a simple task to your Notion backlog. | `/add Buy milk` |
| `/show` | List all current pending tasks from Notion. | `/show` |
| `/done [indices]` | Mark tasks as completed using their index numbers. | `/done 1, 2` |

## 💡 Creative & Academic Suite
*High-fidelity AI research engines that save directly to your Note Vault.*

| Command | Action | Example |
| :--- | :--- | :--- |
| `/note [topic]` | Generate deep, exam-ready academic study notes. | `/note Quantum Physics` |
| `/brainstorm [idea]` | Generate creative execution roadmaps and expansion ideas. | `/brainstorm Start a Podcast` |

## 💰 Financial Tracking & Analytics
*Manage your spending reports and interactive finance logging.*

| Command | Action | Example |
| :--- | :--- | :--- |
| `/budget` | View your monthly spending report by category. | `/budget` |
| `/setbudget [amount]` | Set your monthly spending limit for visual reports. | `/setbudget 30000` |
| `/export` | Export your full financial history to an Excel (.xlsx) file. | `/export` |

## 💪 Habit Tracking & Consistency
| Command | Action | Example |
| :--- | :--- | :--- |
| `/habits` | Open your interactive Daily Consistency Board. | `/habits` |
| `/habitstats` | View deep analytics, streaks, and completion scores. | `/habitstats` |
| `/addhabit [name]` | Register a new habit to start tracking. | `/addhabit Meditation` |

## 🛠️ System & Guidance
| Command | Action |
| :--- | :--- |
| `/status` | View the **Technical Heartbeat** (Latency, Uptime, Worker Health). |
| `/remind [time] [task]`| Set a precise reminder (e.g., 10m, 1h, or at 17:30). |
| `/time` | Run clock diagnostics and check your current timezone. |
| `/tz [timezone]` | Manually set your local timezone (e.g., `Asia/Kolkata`). |
| `/start` | Launch the main interactive welcome dashboard. |
| `/help` | Display the comprehensive guide to Axiom OS features. |

---

## 🎙️ Multimodal Features (No Commands Needed)
Axiom OS supports natural interactions without needing a specific command:

- **Voice Brain Dump**: Send a voice note for a task (*"Remind me to..."*) or an expense (*"Spent 500 on..."*). The bot uses Gemini AI to determine your intent.
- **Natural Finance**: Simply type "Dinner 450" or "Gas 2500" to trigger the interactive categorization flow.

## 💡 Pro-Tips
- **Bulk Completion**: Use `/done 1, 2, 3` to clear multiple tasks at once.
- **Technical Monitoring**: If the bot feels slow, run `/status` to check your current **Notion Latency**.
- **PDF Generation**: After generating a `/note`, you can click the `📥 Get PDF` button to download a formatted document.
