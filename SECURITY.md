# Security Policy

## Supported Versions

Currently, only the latest version of Axiom OS is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, please report it via email:

**Email**: [vk2207419@gmail.com]

We will investigate all reports and attempt to fix vulnerabilities as quickly as possible.

## Personal Responsibility
Axiom OS deals with sensitive API keys (Gemini, Telegram, Notion). Users are responsible for keeping their `.env` files secure and never committing them to public repositories.
